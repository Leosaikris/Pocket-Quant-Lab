# Pocket Quant Lab

A cross-platform ML-driven quantitative research engine for real-time market modeling, systematic strategy development, and disciplined backtesting.

---

## 🚀 Vision

Build a portable quantitative research lab that enables:

- Real-time market signal generation
- Systematic strategy development
- Robust backtesting with realistic execution modeling
- Portfolio-level risk evaluation
- Cross-platform accessibility (browser-first architecture)

---

## 🧠 Core Philosophy

This is not a trading bot.

This is a research engine.

The system emphasizes:
- Deterministic signal logic
- Time-series aware validation
- No lookahead bias
- Disciplined performance evaluation
- Modular architecture

---

## 🏗 Architecture Overview

The engine is divided into:

- Research Core
  - Data Module
  - Feature Engine
  - Strategy Engine
  - Simulation Engine
  - Evaluation Engine

- Portfolio Layer
- Runtime Layer
- API Layer
- UI Layer
- Deployment Layer
